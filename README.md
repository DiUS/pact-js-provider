Node Provider for Pact
=======

Matches pact for Providers. See PACT for details.

Setup
=======

You will need Ruby 2.1.x. You can install this via ruby-install, chruby, .cat-version....
Then run
```
gem install bundler
```

Pact will on preinstall run bundle install
